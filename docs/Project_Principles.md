# SaveCloud Principles

1. Steam is the primary frontend.

2. SaveCloud owns the canonical copy.

3. Games only use working copies.

4. SaveCloud is storage backend agnostic.

5. Save synchronization should require minimal user interaction.

6. Never overwrite without backup.

7. Every sync should be recoverable.

8. Manual configuration is preferred over incorrect automation.

9. Device-specific information should never be synchronized.

10. The system should continue functioning if any storage backend is unavailable.
